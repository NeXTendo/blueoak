# BlueOak Icons

Replace all .png files in this directory with actual BlueOak brand icons.

Required files:
- favicon.ico        (32x32)
- favicon-16.png     (16x16)
- favicon-32.png     (32x32)
- apple-touch-icon.png (180x180)
- icon-72.png
- icon-96.png
- icon-128.png
- icon-144.png
- icon-152.png
- icon-192.png       ← REQUIRED by PWA spec
- icon-384.png
- icon-512.png       ← REQUIRED by PWA spec
- icon-512-maskable.png  ← Design with safe zone (80% visible area)

Generate all sizes from a 1024x1024 master SVG using:
  https://www.pwabuilder.com/imageGenerator
